This is Chukchi, the free Web-based RSS aggregator.

Our goals are to be simple and friendly. Currently the codebase of Chukchi is under 800 lines of Python code and under 500 lines of Javascript code.

If you like Chukchi, please remember, that we are an open source project under GNU AGPL, and you are very welcome to run our code on your own site. This way you can be sure that your RSS aggregator will never get closed by an evil benevolent international corporation.

We would also appreciate your help very much, especially if you're proficient in Python, Javascript, Web design, or just have some good ideas, or are simply a pleasant person to talk to. Just write us at chukchi-project@hades.name.

Please report any problems with this site, or with your instance of the site using Github's issue tracker here.
